# XMB-FCAMOD Theme for ES-DE / FCAMOD

![Platform](https://img.shields.io/badge/Platform-ES--DE%20%2F%20FCAMOD-purple)
![Style](https://img.shields.io/badge/Style-XMB-blue)

A premium, high-performance XMB-styled theme designed specifically for **ES-DE (EmulationStation Desktop Edition)** and the **FCAMOD** fork. This theme brings the classic console interface experience to your handheld devices with smooth animations and a refined layout.

![XMB-FCAMOD Preview](./xmb.png)

## 🌟 Features

- **Classic XMB Interface**: Faithful recreation of the XMB (Cross Media Bar) experience found on PSP/PS3.
- **Multi-Aspect Ratio Support**: Optimized layouts for different screen formats:
  - **4:3**: Perfect for R36S, RG351MP, and other standard handhelds (640x480).
  - **1:1**: Tailored for square screens like the RGB30 (720x720).
- **Violet PSP Aesthetics**: Features a sleek, dark violet color palette inspired by the PSP system.
- **Dynamic Backgrounds**: Support for both static image waves and animated video backgrounds (`waves.mp4`).
- **High-Quality Typography**: Uses *FOT-NewRodin Pro* fonts for a premium, native feel.
- **Optimized for FCAMOD**: Leverages advanced grid layouts and video delay features supported by the FCAMOD engine.

## 📂 Installation

1. Connect to your device via SCP or SFTP.
2. Navigate to your themes directory (usually `/roms/themes/` or `~/.emulationstation/themes/`).
3. Copy the `XMB-FCAMOD` folder into the themes directory.
4. In EmulationStation, go to **UI Settings** > **Theme Set** and select `XMB-FCAMOD`.
5. Under **Aspect Ratio**, select the one that matches your device (4:3 or 1:1).
